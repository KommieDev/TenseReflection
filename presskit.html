In development since 2016, Tense Reflection is an innovative puzzle / shooter where you need to solve puzzles in order to reload your ammo / change the color of your shots.

You must quickly switch between 3 panels :

The Color Panel : To select the color of your shots

The Puzzle Panel : To apply the selected color

The Shooting Panel : Enemies are constantly progressing !
Developer : Kommie

Genre : Puzzle, Shooter

Platforms : Windows PC, MAC OS, Linux

Release : August 2019

Features : Single-Player Campaign, Level Editor and Workshop
